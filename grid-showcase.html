Novo Layout (show.php)

.grid-layout-container { display: grid; grid-template-columns: 1fr 350px; grid-gap: 2rem; max-width: 1400px; margin: 0 auto; } .grid-sidebar { position: sticky; top: 120px; height: fit-content; } @media (max-width: 1024px) { .grid-layout-container { grid-template-columns: 1fr; } .grid-sidebar { position: static; display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); } }

Vantagens

✅ Sidebar Sticky: Fica visível ao scroll em desktop
✅ Responsivo: Automaticamente adapta em tablets e mobile
✅ Semântico: Usa nomes de classe descritivas
✅ Performance: Sem JavaScript, puro CSS
✅ Flexível: Fácil de customizar proporções
